# 从人工神经元到聊天机器人

## 📖 序言

深度学习的历史可以追溯到1943年，神经科学家麦卡洛克（McCulloch）与逻辑学家皮茨（Pitts）提出的**M-P神经元模型**。此后历经“深度学习三巨头”：辛顿（Hinton）、杨立昆（LeCun）、本吉奥（Bengio）等对算法荒原的数十载耕耘，一直至2017年Google发表”Attention is All You Need“，正式确立了**Transformer架构**，开启了**LLM大语言模型**的狂飙时代。

在今天深度学习的领域，以PyTorch和TensorFlow为代表的模型训练框架占据了绝对主导。它们极大地降低了实践门槛，让深度学习得以快速普及。但是，这种便利的代价是将核心逻辑封装进“黑盒”之中，使模型开发在某种程度上变成了屏蔽底层原理的调参比赛。

> “穷理以致其知，反躬以践其实。”
> 
> 《宋史：朱熹传》

对深度学习真正的理解不应止步于调库（import）和调参（tuning）。

## 📖 项目

本项目的宗旨在于抛开框架，直面算法。从最基础的全连接人工神经元网络起步，仅使用NumPy，手工构建每一行逻辑，在实践中理解深度学习的核心思想：

**🌱 起点：** 从实现一个最简单的人工神经元网络模型开始，逐渐扩展到多层感知机（MLP），理解链式法则。

**🛠️ 升级：** 实现一个支持自动微分（Autograd）和反向传播机制的类PyTorch框架，理解张量运算的底层逻辑。

**🏗️ 演进：** 继续扩展框架功能，复现CNN、RNN、LSTM等经典模型，掌握处理图像数据、文本数据的不同思路。

**🤖 涌现：** 从RNN衔接至Transformer架构，完成一个微型聊天机器人（Mini-Chatbot），亲见智能的涌现。

所有章节采用Jupyter Notebook完成，集代码和介绍于一体。每个章节的代码完整，均可独立执行。所有代码力求简洁，聚焦核心功能。所有数据集力求最小化，可以在普通电脑完成模型训练。

## 📖 预备知识

* 微积分基础知识：可以理解微分梯度下降算法、链式规则等。
* 线性代数基础知识：可以理解矢量和矩阵运算。
* Python基本知识：可以使用NumPy完成矢量计算。
* 机器学习基本知识：可以理解线性回归、分类等基本算法。
